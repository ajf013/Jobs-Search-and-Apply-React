# Jobs Search and Apply React

[![Netlify Status](https://api.netlify.com/api/v1/badges/ed96682d-a518-473f-9738-ff2878101f35/deploy-status?branch=main)](https://app.netlify.com/projects/jobssearchandapply/deploys)

A modern React application for searching and applying for jobs.

### [Live Site](https://jobssearchandapply.netlify.app/)

## 🚀 Features

- **Job Search**: Search for jobs by title, expertise, or location.
- **Job Details**: View detailed job descriptions and application instructions.
- **Responsive Design**: Built with React Bootstrap and Semantic UI for a seamless experience on all devices.
- **Animations**: Smooth scroll animations using AOS.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: React Bootstrap, Semantic UI React
- **HTTP Client**: Axios
- **Utilities**: React Markdown (for rendering job descriptions)

## 📦 Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ajf013/Jobs-Search-and-Apply-React.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Jobs-Search-and-Apply-React
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
